# speedtest_ms
MicroService with Nameko. RabbitMQ and SpeedTest

## Run

```bash
docker compose up --build
```

## Test

```bash
pytest
```

## Coverage

```bash
pytest --cov=./src --cov-report=xml
```

## Run with Nameko

```bash
python run.py
```

## Run with Docker

```bash
docker compose up --build
```
